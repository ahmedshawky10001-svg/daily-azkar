## عن المشروع

هذا المشروع هو تطبيق ويب بسيط يهدف إلى تسهيل قراءة الأذكار اليومية بشكل مرتب وسهل الوصول، مع عداد تفاعلي يساعد على إتمام عدد مرات كل ذكر.

### ما يقدمه المشروع

- **أذكار منظمة**: أذكار الصباح، المساء، النوم، الاستيقاظ، والورد اليومي
- **عداد تفاعلي** لكل ذكر مع إمكانية إعادة العد
- **تصميم متجاوب** يعمل على جميع أحجام الشاشات
- **واجهة هادئة** بتصميم بسيط يركز على المحتوى
- **سريع** — لا يحتاج لخادم خلفي
